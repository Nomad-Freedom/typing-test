This game was built with React, TypeScript, and CSS.

## Installation

1. Download repository
   ```bash
   git clone https://github.com/Nomad-Freedom/typing-test.git
   ```
2. Install packages
   ```bash
   npm install
   ```

## Run in Development

1. run application
   ```bash
   npm run dev
   ```
